#docker-base-zh
==============

##为中国用户准备的基础Docker镜像

###本镜像包含以下内容：
* 基于 phusion/baseimage:0.9.16 (Ubuntu 14.04)
* 时区设置为 +8 区
* 软件源设置为 [mirrors.aliyun.com](http://mirrors.aliyun.com)
* 包含 [Ansible](http://docs.ansible.com/) 最新稳定版本（构建时）

###使用方法
请参考 [phusion/baseimage](https://github.com/phusion/baseimage-docker)
